# 竞品分析：AI 生成式学习工具赛道

## 1. 调研范围

| 竞品 | 类型 | 核心差异 |
|------|------|---------|
| AI牛马刷题 | 生成式刷题工具 | 材料输入→单题流式生成→即时反馈。缺RAG/间隔重复/多格式 |
| Anki | 传统间隔重复 | 手动制卡，无AI生成。学习科学最强但制卡成本极高 |
| Quizlet | 社交闪卡 | AI生成卡片+社交共享。偏语言学习，不强调长期记忆 |
| Notion AI | 笔记+AI问答 | AI可对笔记提问。但不生成结构化题目，无间隔重复 |
| Knowt | AI学习助手 | AI从笔记生成题目+间隔重复。最接近RecallForge |

## 2. 详细对比

| 维度 | RecallForge | Knowt | AI牛马刷题 | Anki |
|------|-----------|-------|-----------|------|
| AI出题 | 批量5题 | 逐题 | 逐题流式 | 无 |
| RAG知识库 | ChromaDB | 有 | 无 | 无 |
| 间隔重复 | SM-2 | 有 | 无 | SM-2 |
| 多格式输入 | 文本/URL/PDF/MD | PDF/笔记 | 纯文本 | 手动 |
| 出题设置 | 五维配置 | 有限 | 无 | 无 |
| 多选支持 | 单选/多选/混合 | 单选 | 单选 | 无 |
| 测验模式 | 全部做完再看 | 无 | 即时反馈 | 无 |
| 部署 | 自建全栈 | SaaS | Vercel | 本地 |

## 3. 差异化总结

1. **批量出题 vs 逐题出题** — 竞争品都是一次一题等API，RecallForge一次5题缓存前端
2. **完整出题设置** — 竞争品无设置或极简，RecallForge五维可配置
3. **测验模式** — 竞争品全即时反馈，RecallForge支持"全部做完再看"模拟考试
4. **多题型混合** — 竞争品只出单选，RecallForge支持单选/多选/混合
5. **RAG+SM-2双引擎** — Knowt有类似组合，但RecallForge的配置自由度更高

## 4. 弱势和风险

- **品牌知名度为零** — 无法和 Quizlet/Anki 的千万用户基础竞争
- **无社交功能** — 无法像 Quizlet 那样通过社区内容获得增长
- **依赖 DeepSeek API** — AI 出题质量和速度受限于第三方服务
- **无移动端 App** — 当前只有 Web，无法覆盖碎片化学习场景
