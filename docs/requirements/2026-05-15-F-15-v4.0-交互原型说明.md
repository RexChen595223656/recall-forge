# v4.0 交互原型说明

## 页面总览

单页应用，两栏布局（左栏知识库 256px + 右栏主操作区），顶部 TopBar。

核心交互节点 7 个：
1. 材料卡片（上传后 + 出题中 + 就绪）
2. 空闲态（题目库模式）
3. 答题中
4. 答题结果
5. 错题重做
6. 间隔复习
7. 全局复习

---

## 节点 1：材料卡片状态

### 状态 A：刚上传（出题中）

```
┌─────────────────────────┐
│  材料标题                │
│  ⏳ 出题中...           │   ← 文案+动画，不显示具体进度
└─────────────────────────┘
```

点击 → 进入空闲态，题目区显示已有题目 + "出题中"提示。

### 状态 B：出题完成

```
┌─────────────────────────┐
│  材料标题                │
│  12 题可用 · 正确率 45%  │
│  ████████░░  近期 80% ↑ │   ← 掌握度条
└─────────────────────────┘
```

### 状态 C：出题失败

```
┌─────────────────────────┐
│  材料标题                │
│  ⚠ 出题失败 · 点击重试   │   ← 可点击
└─────────────────────────┘
```

## 节点 2：空闲态（题目库模式）

```
┌────────────────────────────────────────┐
│  材料标题                  总 45%      │
│  ████████░░░░░░░░░░      近期 80% ↑   │
│  12 题可用 · 已覆盖 3/5 知识点          │
│                                        │
│  [原文]                                │
│                                        │
│  ── 选题设置 ──                        │
│                                        │
│  知识点（多选）                         │
│  [不限] [架构] [实践] [概念]            │
│                                        │
│  数量    [5] [10] [全部(12)]           │
│  模式    [原文] [拓展]                  │
│  反馈    [即时] [测验]                  │
│                                        │
│  [开始答题]          [补充题目]         │
│                                        │
│  ── 巩固 ──                            │
│                                        │
│  🔴 错题重做 · 3题待攻克    [开始 →]    │
│  🔵 间隔复习 · 2题到期      [开始 →]    │
│                                        │
│  ── 全部题目 ──                        │
│  ▶ 全部题目（12题）                    │
│    └ 展开后显示每道题 + 答案           │
│                                        │
│  ── 最近答题 ──                        │
│  ▶ 05-15 14:30 正确 3/5              │
│  ▶ 05-15 13:50 正确 4/5              │
│                                        │
│  ── 知识标签 ──                        │
│  [架构:5] [概念:3] [实践:4]            │
└────────────────────────────────────────┘
```

### 知识点筛选交互

```
点击 [架构] → 高亮选中
  ↓
[不限] 自动取消高亮
全部题目区过滤为架构:5 的题目
数量选项：5/10 → 5（因为架构仅 5 题）

再点击 [实践] → 同时选中架构 AND 实践
  ↓
全部题目区过滤为同时有架构+实践标签的题目
数量选项动态调整

点击 [不限] → 清除所有筛选
```

### 补充题目交互

```
点击 [补充题目]
  ↓
按钮变为 [补充中...]（disabled）
后端异步从新 chunk 出题
  ↓ 完成后
按钮文案更新，题目库数量增加
如果 covered = total → [已无新题可出]（disabled）
```

## 节点 5：错题重做

```
┌────────────────────────────────────────┐
│  [← 返回]  错题重做          2 / 3     │
│                                        │
│  以下哪项不是 AI PM 的核心能力？        │
│                                        │
│  [A. 了解模型训练成本]                   │  ← 用户选了
│    ← 你的选择（红色）                    │
│  [B. 知道模型能做什么]                   │
│  [C. 设计 Prompt 系统]                  │
│  [D. 建立评估体系]                      │
│                                        │
│  ✗ 回答错误                            │
│                                        │
│  [下一题]                              │
├────────────────────────────────────────┤
│  streak: ▯▯▯  (0/3)           ← 进度  │
└────────────────────────────────────────┘
```

### 答对时

```
│  [A. ...]                              │
│  [B. 知道模型能做什么]  ← 你的选择       │
│    ← 正确（绿色）                        │
│                                        │
│  ✓ 回答正确                            │
│                                        │
│  streak: ▮▯▯  (1/3)                   │
```

### 攻克动画

```
连续 3 次做对时：
  ↓
显示 "✓ 已攻克！"（绿色大字，0.5 秒）
  ↓
自动跳转下一题
该题从待攻克列表移除
```

## 节点 6：间隔复习

与 v3.x SM-2 复习模式相同：

```
┌────────────────────────────────────────┐
│  [← 返回]  间隔复习          1 / 2     │
│  复习 3 次 · 连续正确 1 次             │
│                                        │
│  以下哪项不是 AI PM 的核心能力？        │
│                                        │
│  [A. 了解模型训练成本]                   │
│  [B. 知道模型能做什么]                   │
│  [C. 设计 Prompt 系统]                  │
│  [D. 建立评估体系]                      │
│                                        │
│  [回忆一下，点击查看答案]                │
│                                        │
│  ── 点击后 ──                          │
│                                        │
│  [A] [B ✓] [C] [D]     ← 答案高亮      │
│                                        │
│  你的记忆程度：                         │
│  [忘记了] [有印象] [记住了] [很简单]     │
│  [跳过]                                │
└────────────────────────────────────────┘
```

### 毕业提示

```
评分后 interval > 21 天 且 repetitions >= 3 →
  底部短暂提示 "此卡片已稳定，移出到期列表"
  ↓
  自动跳转下一题
```

## 补充视图

### 已攻克列表（点击"查看已攻克列表"后）

```
┌────────────────────────────────────────┐
│  [← 返回]  已攻克题目                  │
│                                        │
│  ✓ 题目内容...(已攻克)                  │
│  ✓ 题目内容...(已攻克)                  │
│  ...                                   │
│                                        │
│  共 5 题已攻克                         │
└────────────────────────────────────────┘
```

每道已攻克题可点击展开查看答案和解析。无需操作。

### 稳定卡片列表（点击"查看稳定卡片"后）

```
┌────────────────────────────────────────┐
│  [← 返回]  稳定卡片                    │
│                                        │
│  题目内容...                           │
│  间隔 35 天 · 连续正确 4 次             │
│  [手动复习]                            │
│                                        │
│  共 3 张稳定卡片                        │
└────────────────────────────────────────┘
```

手动复习后卡片回到活跃队列（is_stable=false, interval=1）。

### 错题重做完成（本轮结束 vs 全部攻克）

**本轮完成（还有剩余）：**
```
┌────────────────────────────────────────┐
│  本轮完成                              │
│  正确 4/5 · 已攻克 +1 题              │
│  还有 3 题待攻克                       │
│  [继续重做] [返回]                     │
└────────────────────────────────────────┘
```

**全部攻克（无剩余）：**
```
🎉 全部错题已攻克！
[查看已攻克列表] [返回]
```
