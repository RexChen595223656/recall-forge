# 知锻 RecallForge 功能拆解说明书 v2.0

## 1. 功能清单（按模块）

### 知识库
| 功能 | 触发 | 流程 | 状态处理 |
|------|------|------|---------|
| 上传材料 | 点击"上传材料" → 选择模式 | 文本/URL/文件 → 后端解析 → 分块 → ChromaDB | 空态:引导文案；加载态:进度条+说明；错误:红色提示 |
| 材料列表 | 进入首页 | 按时间倒序展示，每项显示格式图标/标题/日期 | 空态:大图标+"知识库还是空的"；hover 显示刷题/删除 |
| 删除材料 | 点击删除 → 确认 | 级联删除:ReviewCard→Attempt→Question→Material+Chunks | 确认弹窗防误删 |
| 示例导入 | 空态点"试试示例" | 预置 AI PM 方法论文本 → 上传 → 跳转刷题 | — |

### 出题
| 功能 | 触发 | 流程 | 状态处理 |
|------|------|------|---------|
| 五维设置 | 点击材料→刷题 | 提取方式/难度/题型/数量/反馈模式 | 默认值:原文提取+中等+单选+5题+即时反馈 |
| 题型选择 | 勾选单选/多选 | 至少保留一个；两者勾选→混合出题 | 复选框样式，选中显示 ✓ |
| 批量生成 | 点击"开始出题" | 后端查已用chunk→取新chunk→DeepSeek生成→保存DB | 生成中:动画+取消按钮；超时45s→错误提示/重试 |
| 回答 | 点击选项 | 单选:即时判定；多选:勾选后确认 | disabled 防重复提交 |
| 即时反馈 | 答完即刻 | 展示:正确/错误+题目回顾+你的选择vs正确答案+解析 | 绿色边框=正确，红色=错误 |
| 测验模式 | 全部答完 | 回顾页:逐题对照答案；异步提交所有答案 | 错题自动加入复习 |

### 复习
| 功能 | 触发 | 流程 | 状态处理 |
|------|------|------|---------|
| 到期复习 | 进入复习页（默认） | 查询 next_review ≤ now 的卡片 | 空态:"暂无到期卡片"+"查看全部"引导 |
| 全部复习 | 切换"全部卡片"标签 | 查询所有卡片，先到期排前 | 未到期卡片显示黄色时间标签 |
| 评分 | 揭晓答案 → 评分 | SM-2:0=忘记(重置)/1=有印象(1天)/2=记住(3天)/3=很简单(拉长) | 每个按钮显示后果提示 |
| 跳过 | 点"跳过，稍后再评" | 跳过当前卡片，不更新 SM-2 | — |
| 完成 | 全部复习完 | 如果是仅到期模式→提示可查看全部；全部模式→可再来一遍 | — |

## 2. AI 功能详细逻辑

### 出题 Prompt 结构

根据五维设置动态构建：

```
模式 → 原文提取指令 / 知识拓展指令
难度 → 简单/中等/困难 选项设计指令
题型 → 单选(4选1) / 多选(5选2-3) / 混合
学习材料 → RAG 检索的 Top-K chunk
输出格式 → JSON 数组（含 question/options/answer/explanation/tags）
```

### 答案标准化

- LLM 可能返回 `"answer": "A"` 或 `"answer": "A. 完整选项"`
- 后端保存时提取首字母标准化
- 前端比较时用 `正确答案.startsWith(用户选择)`

## 3. 状态矩阵

| 功能 | 正常态 | 空态 | 加载态 | 错误态 | 兜底 |
|------|--------|------|--------|--------|------|
| 材料上传 | 上传成功跳转刷题 | 引导文案+示例按钮 | 进度条+说明文字 | 红色错误提示 | — |
| AI出题 | 批量返回题目 | — | 动画+取消按钮 | 重试/返回设置 | 超时45s自动报错 |
| 答题 | 选项可点击 | — | — | — | 本地判定兜底 |
| 复习 | 卡片+评分 | 引导查看全部 | 骨架屏 | — | — |
| 生成中 | 动画 | — | 动画 | 取消→返回设置 | 超时→错误提示 |

## 4. 数据关系

```
Material (1) → (N) Question → (N) Attempt
                              → (1) ReviewCard
删除 Material → 级联删除所有关联数据
```
