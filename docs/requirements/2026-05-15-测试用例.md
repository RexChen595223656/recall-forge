# v3.0 锻造台重构 测试用例

## 1. 知识库边界用例

| 编号 | 场景 | 输入 | 预期结果 |
|------|------|------|---------|
| K-01 | 空知识库 | 无材料 | 左栏:空态引导；右栏:欢迎引导 |
| K-02 | 单条材料 | 仅 1 条材料 | 卡片正常显示，无异常间距 |
| K-03 | 大量材料(>20) | 20+ 条材料 | 左栏可滚动，无性能问题 |
| K-04 | 搜索无匹配 | 搜索"xyz" | 显示"无匹配材料" |
| K-05 | 搜索特殊字符 | 搜索"<script>" | 不执行脚本，正常过滤 |
| K-06 | 筛选"薄弱" | 全部掌握度≥50% | 列表为空，提示无薄弱材料 |
| K-07 | 材料标题超长 | 100字标题 | 卡片内截断显示，不撑破布局 |
| K-08 | 删除当前选中材料 | 锻造区正在展示该材料 | 锻造区回到全局概览，左栏移除该卡片 |
| K-09 | 删除最后一条材料 | 仅 1 条材料 | 列表回到空态，无控制台报错 |

## 2. 锻造区边界用例

| 编号 | 场景 | 输入 | 预期结果 |
|------|------|------|---------|
| F-01 | 材料无任何数据 | 选择"未开始"的材料 | 掌握度 0%，灰色进度条，无历史记录 |
| F-02 | 材料全部正确 | 正确率 100% 的材料 | 掌握度显示 100%（含加权），绿色进度条 |
| F-03 | 材料全部错误 | 正确率 0% 的材料 | 掌握度 0%，红色进度条，大量待复习 |
| F-04 | chunk 全部用过 | 该材料所有 chunk 都出过题 | 提示"该材料已无新题可出"，建议上传更多 |
| F-05 | chunk 不足 | 材料很短，只有 1-2 个 chunk | 正常出题（允许重复使用 chunk） |
| F-06 | 生成中取消 | 点击"取消生成" | 回到空闲态，不产生任何题目 |
| F-07 | 生成超时 | 模拟网络延迟 > 45s | 显示"AI 响应超时"+ 重试按钮 |
| F-08 | AI 返回空数组 | 后端返回 `{"questions":[]}` | 显示"未能生成题目" + 返回按钮 |
| F-09 | AI 返回格式异常 | 后端返回非 JSON 文本 | 显示"AI 返回格式异常" + 返回按钮 |
| F-10 | 答完全部正确 | 5 题全对 | 结果页：5/5，无加入复习的题目 |
| F-11 | 答完全部错误 | 5 题全错 | 结果页：0/5，5 题全部加入复习 |
| F-12 | 答题中途关闭标签页 | 浏览器关闭 | 已保存的题目不丢失，已提交的答案已记录 |

## 3. 答题交互边界用例

| 编号 | 场景 | 输入 | 预期结果 |
|------|------|------|---------|
| A-01 | 快速双击选项 | 0.1s 内双击同一选项 | 只触发一次判定 |
| A-02 | 快速连续点不同选项 | 点 A 后立刻点 B | 第一次点击后按钮 disabled，第二次无效 |
| A-03 | 答案首字母大小写 | LLM 返回 "b" | 后端标准化为 "B"，判定正常 |
| A-04 | 答案带点号 | LLM 返回 "B." | 后端标准化为 "B"，判定正常 |
| A-05 | 答案带完整文本 | LLM 返回 "B. Agent架构" | 后端标准化为 "B"，判定正常 |
| A-06 | 多选全部勾选 | 5 个选项全选 | answers 比较不符合 → 判定错误 |
| A-07 | 多选一个不选点确认 | 未勾选任何项 | 确认按钮 disabled，无法提交 |
| A-08 | 选项不带字母前缀 | 选项文本"直接是内容" | charAt(0) 取"直"，不会崩溃 |
| A-09 | 选项为空字符串 | options 中含 "" | 不导致页面崩溃 |

## 4. 复习边界用例

| 编号 | 场景 | 输入 | 预期结果 |
|------|------|------|---------|
| R-01 | 无到期卡片 | 该材料所有卡片 next_review > now | 显示"暂无待复习的题目" |
| R-02 | 仅 1 张到期卡片 | 1 张卡片 | 评分后→完成页 |
| R-03 | 连续评分"忘记了" | 多次 score=0 | 间隔始终为 1 天，不增大 |
| R-04 | 连续评分"很简单" | 多次 score=3 | 间隔逐渐增大（1→3→...） |
| R-05 | 评分后立刻关闭 | 评分→不等待保存→关闭 | SM-2 已更新（record 已调） |
| R-06 | 跳过所有卡片 | 每张都点"跳过" | SM-2 不更新，卡片仍在队列 |
